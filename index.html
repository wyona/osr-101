About

The Neutron Protocol (NP) is an XML-based application-level protocol to abstract the content management user interface from the server side implementation of a content management system respectively framework. A graphical overview can be found here. The Neutron Protocol is closely related to WebDAV and APP.

  2. Make introspection documents available as referenced by the auto-discovery link
  3. Implement open and save functionality as referenced by the introspection documents
